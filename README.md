# MESD: Enhancing Multimodal Science Question Answering via Masked and Ensemble Self-Distillation

This repository contains the code for the paper "Enhancing Multimodal Science Question Answering via Masked and Ensemble Self-Distillation".

## Abstract
Multimodal science question answering (MSQA) aims to answer scientific questions via a multimodal combination of text, images, and diagrams. Current MSQA works enhance multimodal reasoning and interaction under the premise of complete or abundant data. However, they are constrained by data integrity. Drawing inspiration from data augmentation and self-distillation, we propose Masked and Ensemble Self-Distillation (MESD) through bidirectional and unidirectional knowledge transfers. Specifically, masked modalities are generated by random augmentations and paired with complete modalities to form partially masked pairs. MESD then encourages the fusion component to adaptively learn robust embeddings from the consistent distribution of these partially masked pairs via masked self-distillation with bidirectional knowledge transfer. Additionally, the dual-grained crossmodal fusion component models the modality correlations and explores the restoration of masked modalities to obtain fused multimodal embeddings. MESD ultimately derives candidate results through independent recommendations and treats the final result as its target, encouraging the logit distribution of candidate results to be consistent with that of the final result. Experiments on two datasets demonstrate the superior performance of MESD, and ablation experiments confirm the effectiveness of each component. Data and source code are available at https://github.com/gaoling-code/MESD.

pip install -r requirements.txt

## Datasets

The models are trained and evaluated on two open-source datasets:
- ScienceQA: Available at:
  - [Hugging Face Repository](https://huggingface.co/cooelf/vision_features/tree/main)
  - [Google Drive Link 1](https://drive.google.com/file/d/13B0hc_F_45-UlqPLKSgRz-ALtFQ8kIJr/view?pli=1)
  - [Google Drive Link 2](https://drive.google.com/drive/folders/1w8imCXWYn2LxajmGeGH_g5DaL2rabHev)
- CK12-QA dataset: Accessible at [Textbook Question Answering](https://prior.allenai.org/projects/tqa).


The processed vision features for ScienceQA are available at [huggingfcae vision features](https://huggingface.co/cooelf/vision_features/tree/main). `all-MiniLM-L6-v2` and `unifiedqa-t5-base` can be downloaded at [huggingface sentence-transformers](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2) and [huggingface unifiedqa-t5-base](https://huggingface.co/allenai/unifiedqa-t5-base).


The folder with all related files looks like:

## Usage

To infer with pretrained weights (`results/base_pretrained_scienceqa/`), run `run_eval_scienceqa.sh`.

To train the model by yourself, please run `run_train_scienceqa.sh`.
